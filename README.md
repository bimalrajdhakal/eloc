# eloc
The purpose of this project is to make an interactive nationwide digital address system. 
This application generates an unique alphanumeric 6 character to represent the communication as well as residential address of an user. The proposed system also enables the user to locate them in Google Map, find nearby palaces, navigation, location history, search places and eLoc, share location and other custom features.
